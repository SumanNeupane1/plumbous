PLUMBOUS.COM FANCY WEBSITE
===========================

Files included:
- index.html
- style.css
- script.js

How to host:
1. Go to Netlify.
2. Click Add new site.
3. Choose Deploy manually.
4. Drag this unzipped folder into Netlify.
5. Add plumbous.com under Domain Management.
6. Update GoDaddy DNS using the records Netlify gives you.

Offer email:
All buttons are set to email: sumanneupane009@gmail.com

Important:
This is a static website. The offer button opens the visitor's email app.
If you want an actual web form that sends emails without opening an email app,
you can later connect Netlify Forms, Formspree, or another form backend.
